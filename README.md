# FlashStack

Mobile-first real estate analysis platform.

## Quick Start
```bash
npm install
npm run dev
```

## Documentation
- **Current State**: See `FlashStack-Current-State.md`
- **Development Log**: See `Development-Log.md`

## Tech Stack
- React + Vite
- localStorage (temporary)
- Mobile-first design (375x812px)

## Project Structure
- `/src` - All React source code
- `/archive` - Old prototypes and documentation
- `/reference` - Brand and design assets

## GitHub Setup
To connect this repository to GitHub:
```bash
# If you haven't created a GitHub repo yet:
# 1. Create a new repository on GitHub
# 2. Update the remote URL:
git remote set-url origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# Push your changes:
git push -u origin master
```
